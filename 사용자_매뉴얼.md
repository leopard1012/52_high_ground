# 사용자 매뉴얼

## 1. 앱 소개

`Ground`는 기도 커뮤니티를 위한 모바일/웹 앱입니다. 사용자는 기도 카드 작성, 챌린지 참여, 크루 활동, 모임 참석, 배지 획득 등의 기능을 사용할 수 있습니다.

## 2. 시작하기

### 2.1 로그인

- 이메일과 비밀번호로 로그인합니다.
- Google 계정으로도 빠르게 로그인할 수 있습니다.

### 2.2 회원가입

- 이메일을 입력하고 비밀번호를 설정하여 회원가입합니다.
- 첫 로그인 후 프로필 닉네임과 아바타를 설정할 수 있습니다.

## 3. 주요 화면 안내

### 3.1 홈 탭 (`HomeTab`)

- 다른 사용자의 기도 요청 카드를 확인합니다.
- 기도하면 `count`가 증가하며, 기도한 사용자 목록이 저장됩니다.
- 새 기도 카드를 작성하려면 텍스트 입력 후 등록합니다.
- 카드에는 작성자 정보, 기도 요청 내용, 이모지 및 배경이 표시됩니다.

### 3.2 챌린지 탭 (`ChallengeTab`)

- 진행 중인 챌린지를 확인합니다.
- 일일 미션을 완료하면 진행 상태가 업데이트되고 보상이 지급됩니다.
- 각 챌린지의 총 진행일, 보상 씨앗, 현재 참여도 등을 확인할 수 있습니다.

### 3.3 고민 상담 섹션

- `ChallengeTab` 내부에서 고민을 등록할 수 있습니다.
- 다른 사용자 또는 관리자가 질문에 답변하면 `replies`가 표시됩니다.
- 상태는 `open` 또는 `resolved`로 표시됩니다.

### 3.4 크루 탭 (`CrewTab`)

- 크루 목록과 랭킹을 확인합니다.
- 각 크루의 총 씨앗, 멤버 수, 활동 현황을 볼 수 있습니다.
- 크루 내 채팅 또는 메시지를 통해 소통할 수 있습니다.

### 3.5 모임 탭 (`MeetingTab`)

- 예정된 기도 모임 일정을 확인합니다.
- 모임 정보에는 시간, 장소, 주최자, 설명이 포함됩니다.
- 참석 신청 또는 출석 체크 기능을 사용할 수 있습니다.

### 3.6 나의 탭 (`MyTab`)

- 내 프로필 정보와 보유 배지를 확인합니다.
- 현재 씨앗 개수와 나무 레벨을 실시간으로 확인할 수 있습니다.
- 닉네임, 아바타, 배지 목록 등 개인 데이터를 관리합니다.

## 4. 자주 사용하는 기능

### 4.1 기도하기

- 홈 탭에서 기도 버튼을 누르면 해당 기도 카드의 기도 횟수가 증가합니다.
- 한 번 기도하면 `prayedBy` 목록에 추가되어 중복 기록을 관리합니다.

### 4.2 기도 카드 작성

- 기도 요청 내용을 입력하고 등록합니다.
- 작성된 카드는 다른 사용자에게 실시간으로 노출됩니다.

### 4.3 챌린지 참여

- 챌린지를 선택하고 오늘의 미션을 완료 표시합니다.
- 챌린지 완료는 사용자 개인 진행도로 기록됩니다.

### 4.4 고민 등록 및 답변

- 고민 상담 섹션에서 `새 고민 작성` 버튼을 누릅니다.
- 질문을 입력 후 제출하면 다른 사용자가 답변할 수 있습니다.
- 답변이 달리면 화면에서 즉시 업데이트됩니다.

### 4.5 크루 활동

- 크루 랭킹에서 내 크루 순위를 확인합니다.
- 크루 내 메시지를 보내거나 정보를 확인해 협업합니다.

### 4.6 모임 참석

- 모임 정보에서 참석 버튼을 누릅니다.
- 참석 상태가 `attendances`에 기록됩니다.

## 5. 알림 및 실시간 동기화

- 앱은 Firebase를 통해 실시간으로 데이터를 동기화합니다.
- 다른 사용자가 작성한 기도 카드, 챌린지 업데이트, 크루 활동 등이 즉시 반영됩니다.

## 6. 문제 발생 시

### 6.1 로그인 실패

- 이메일 또는 비밀번호를 확인합니다.
- Google 로그인이 정상 작동하지 않으면 네트워크 상태를 확인합니다.

### 6.2 데이터 표시 오류

- 페이지를 새로고침하여 최신 상태를 확인합니다.
- 문제가 계속되면 앱을 종료 후 재접속합니다.

### 6.3 문의 방법

- 관리자 또는 개발자에게 앱 사용 중 발생한 오류를 알려주세요.
- 에러 메시지, 발생 시간, 사용 중인 화면 정보를 함께 전달하면 빠른 해결에 도움이 됩니다.

## 7. 권장 사용 팁

- 정기적으로 챌린지와 모임 정보를 확인하세요.
- 크루 메시지를 통해 다른 사용자와 협력하세요.
- 모임 참석으로 오프라인 활동을 연결하세요.
- 배지를 모으고 씨앗을 쌓아 레벨을 올리세요.
